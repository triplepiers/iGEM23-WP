# Wiki - HW1

## 1 内容描述

请使用您熟悉的前端技术栈完成一个<u>静态网页</u>。

- 您可以进行原创设计，也可以选择复刻自己喜欢的网页的（部分）内容。后者需要附上被复刻网站的 URL。

- 您需要对 JS / TS 代码进行适当注释；若您认为有必要，也可以对 CSS 进行部分注释。

最终需要提交的内容包括：

- 所有的 Source Code
- 一份说明文档，简要介绍（您认为的）亮点功能实现思路
